# DailyTuring
 A script to get the daily Turing Machine puzzle, then store it in this git, then publish it to the adjacent GitHub pages at [daily-turing.peiphy.xyz](daily-turing.peiphy.xyz).

## GitHub Pages calendar
The GitHub Pages root now shows a generated calendar. Days with complete data are shown normally and are clickable, while days with missing/0-byte data are grayed out.

A day is considered complete only when all 5 files are non-empty:
- `criteria`
- `solution`
- `verifiers`
- `hash`
- `score`
